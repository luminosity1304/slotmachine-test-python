cute slots

its a lil slot thing, cute and silly. you start with $1000 cash.

run it like this:

```bash
python gambling/slot_machine.py
```

controls:
- bet with the spinbox
- press spin 🎰
- bet max = all your cash
- reset = back to 1000

wins:
- 3 same = 5x your bet
- 2 same = 2x your bet
- else you lose your bet

have fun!! 🌸
